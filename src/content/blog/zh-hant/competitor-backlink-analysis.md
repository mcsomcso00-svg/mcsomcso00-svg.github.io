# 競爭對手外鏈分析：用 Ahrefs 找機會

**競爭對手外鏈分析的核心價值：別人已經替你驗證過「誰願意給這個主題連結」。** 與其大海撈針找建鏈機會，不如把對手的 backlink profile 攤開——連過他們的資源頁、媒體、部落格，就是你最該優先接觸的名單；他們靠哪篇內容吸到最多連結，就是你內容策略的直接情報。

## 第一步：選對「對手」

很多人直接分析產業龍頭，結果拿到一張根本追不動的名單。正確的取樣：

- **SERP 對手優先**：跟你搶同一批關鍵字的網站，而不是商業上的直接競品。用 Ahrefs 的 Organic competitors 或直接搜目標關鍵字看前 10 名。
- **選 2–4 個量級相近的**：對方 referring domains 是你的 2–5 倍最理想——差距代表機會，量級相近代表可複製。
- **加 1 個標竿**：一個明顯領先的網站，用來看天花板長什麼樣。

## 第二步：Link Intersect 找「共同來源」

這是整個流程裡投報率最高的一招。Ahrefs → Competitive Analysis（Link Intersect）→ 填入 2–4 個對手網域、你自己的網域放在「but doesn't link to」欄位，跑出來的清單就是：**連了多個對手、卻沒連你的網站**。

這張清單為什麼珍貴？因為連了 3 個對手的網站，等於用行動證明了三次「我願意連這類主題」——你缺的只是一個接觸理由。逐條標記類型：

| 來源類型 | 代表機會 | 動作 |
|---|---|---|
| 資源／清單頁 | 對方在維護推薦清單 | 寄信自薦，說明你補了什麼缺 |
| 產業媒體報導 | 對方會寫這個主題 | 列入數位 PR 名單 |
| 部落格正文引用 | 對方引用了對手的數據或工具 | 做出更好的版本再接觸 |
| 目錄／論壇 | 可自行提交 | 挑有真實流量的做 |

## 第三步：Best by links 找「吸鏈內容」

Site Explorer → 對手網域 → **Best by links** 報表，依 referring domains 排序，你會看到對手全站最會吸連結的頁面。常見的模式：

1. **數據報告**吸走最多媒體連結 → 你的產業缺哪份數據？
2. **免費工具**吸走最多教學文連結 → 有沒有你能做得更好的工具？
3. **終極指南**吸走最多資源頁連結 → 對手的指南更新了嗎？有沒有明顯缺口？

這個報表回答的問題是「這個利基的連結市場買什麼單」，直接決定你該做哪種可引用資產——對應做法見〈[如何建立高品質外鏈](/zh-hant/blog/quality-link-building/)〉。

## 第四步：看新增連結，抓對手的「現在進行式」

Site Explorer → Backlinks → 篩選 **New**，看對手最近 30–90 天拿到的新連結。這能告訴你：對手正在跑什麼活動（客座潮＝在投稿、媒體潮＝在做 PR）、哪些網站「現在」願意給連結——時效性比陳年老鏈高得多。

## 重要：學會辨認「不該抄的作業」

對手的外鏈裡通常混著他們買的連結。照單全收等於替別人的風險買單。付費連結的典型特徵：

- 錨文本是精確商業關鍵字（自然引用極少這樣，原理見〈[錨文本多樣化實戰](/zh-hant/blog/anchor-text-diversity/)〉）。
- 來源站什麼主題都寫、每篇都外連不同產業的商業站。
- 內容是為了塞鏈而寫的 500 字空殼文。

看到這類連結的正確反應是跳過，而不是「對手買了我也買」。對手可能正在累積一筆還沒爆的負債。

## 從分析到執行：一份可操作的輸出

分析的終點是一張行動表，建議欄位：來源網域、連結對手的哪一頁、來源類型、接觸窗口（記者名／站長信箱）、優先級、我方對應資產。每月安排固定時段執行 10–20 個接觸，比一次性衝刺更能累積。沒有對應資產的機會（例如對方連的是對手的工具），先記下來，讓它進入你的內容排程。

## 常見問題（FAQ）

**Q1：沒有 Ahrefs 預算，有替代方案嗎？**
有。Semrush 的 Backlink Gap、Moz Link Explorer 功能類似；Google Search Console 只能看自己的連結，做不了對手分析。Ahrefs 也有功能受限的免費 Webmaster Tools，至少能把自己這端看清楚。

**Q2：對手的外鏈數量是我的 20 倍，追得上嗎？**
不需要追上總量。排名比的是「相關頁面層級」的競爭力：你的目標頁面加上站內權重傳遞，對上對手那一頁的連結，差距通常遠小於全站數字。集中火力在少數目標頁比全面開戰實際。

**Q3：Link Intersect 清單裡的網站，聯絡成功率大概多少？**
低是常態，一成上下的回覆率就算健康。所以名單品質（對方真的在維護內容）和接觸理由（你真的補了缺口）比寄信量重要，群發模板信只會訓練對方忽略你。

**Q4：多久做一次競爭對手外鏈分析？**
完整分析每季 1 次足夠；New backlinks 的監控可以設成每月例行。外鏈格局變化沒那麼快，過度頻繁的分析只是拖延執行的藉口。
