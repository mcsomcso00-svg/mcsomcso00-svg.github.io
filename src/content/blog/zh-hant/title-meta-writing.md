# 標題標籤與 meta description 怎麼寫

**title 標籤是排名訊號，寫法核心是「關鍵字前置＋講清楚頁面內容」，繁中建議 30 字內；meta description 不是排名因素，它的任務是提高點閱率，用 80–120 字寫出這頁能解決什麼問題。** 兩者是搜尋結果頁上你唯一能控制的廣告文案。

## 兩者的分工：一個給演算法，一個給人

| 項目 | title 標籤 | meta description |
|---|---|---|
| 是排名因素嗎 | 是（Google 官方確認的輕量訊號） | 否（官方多次確認不影響排名） |
| 主要任務 | 告訴 Google 這頁在講什麼 | 說服搜尋者點你而不是點別人 |
| 顯示長度 | 桌機約 600px（繁中約 30 字） | 約 2 行（繁中約 80 字內完整顯示） |
| 被 Google 改寫的機率 | 高（官方說明會依查詢重寫） | 更高（經常直接抓內文片段） |

## title 標籤的寫法規則

1. **核心關鍵字放前面**：越前面的詞，權重訊號與使用者掃視的注意力都越高。「內容負債是什麼？90 天 0 點擊頁面正在拖垮你」優於「你的網站正在被一種叫內容負債的東西拖垮」。
2. **長度控制在 30 個中文字內**：Google 依像素寬度截斷（桌機約 600px），中文字寬，超過就變「…」。截斷不會被處罰，但重要資訊要在前段講完。
3. **一頁一個主關鍵字**：一個 title 想塞三個關鍵字，通常三個都排不好——這是主題群集規劃的問題，不是標題技巧問題（見〈[主題群集怎麼規劃](/zh-hant/blog/topic-cluster-planning/)〉）。
4. **加差異化元素**：數字（「30 個必查項目」）、年份（2026）、具體承諾（「附操作」「附範例」）。搜尋結果上你和九個對手並排，一模一樣的標題沒有點的理由。
5. **品牌名放尾端、可省略**：「文章標題｜品牌名」是標準格式；長尾文章空間不夠時可拿掉品牌名。
6. **每頁唯一**：全站重複的 title 是最常見的技術債，用爬蟲工具（Screaming Frog 免費版）一次抓出來。

三個可直接套的模板：

- 定義型：`[關鍵字]是什麼？[一句話價值主張]`
- 教學型：`[關鍵字]怎麼做？[數字]個步驟（附[範例/清單/模板]）`
- 清單型：`[數字]個[關鍵字][錯誤/工具/技巧]，[痛點或承諾]`

## meta description 的寫法規則

1. **80–120 字，前 80 字講完重點**：行動版顯示更短，關鍵訊息前置。
2. **寫「這頁替你解決什麼」，不是「這頁包含什麼」**：「三步驟用 GSC 找出拖垮排名的 0 點擊頁面，附實際報表名稱與操作路徑」優於「本文介紹 GSC 的使用方法」。
3. **包含關鍵字**：不為排名，而是因為 Google 會把查詢詞**加粗**顯示——粗體字提高被看見的機率。
4. **放具體元素**：數字、方法名、可交付物（清單、模板、流程圖）。
5. **每頁唯一、與內文相符**：描述說有「價格比較表」內文卻沒有，換來的是跳出。

## Google 改寫你的標題怎麼辦？

自 2021 年的標題改寫更新後，Google 會依查詢語境替換顯示標題，官方數據約有 **13%** 的情況不用你的 title 原文。常見觸發原因與對策：

- **title 太長被截** → 縮短到重點前置。
- **關鍵字堆砌** → 改寫成一句自然的話。
- **title 與 H1、內文不一致** → 讓 title 與 H1 保持高度一致（可以不同，但主題要相同）。
- **全站樣板化標題** → 每頁手寫差異化。

改寫無法完全避免，但以上四點能把機率壓到最低。

## 用 GSC 找出「該改標題」的頁面

這是最被低估的免費優化：

1. 進 GSC「成效 → 搜尋結果」，日期設 90 天，勾選顯示「曝光」與「點閱率」。
2. 切到「網頁」分頁，找出**曝光高但 CTR 明顯偏低**的頁面（例如曝光 > 1,000、CTR < 1%）。
3. 點進該頁看「查詢」分頁——使用者實際用什麼詞搜到你，title 有沒有回應那個意圖？
4. 改寫 title 與 description，記錄日期，4 週後回來比對 CTR。

這招對「有曝光、0 點擊」的輕度內容負債頁特別有效（負債判定見〈[GSC 三個地方揪出內容負債](/zh-hant/blog/gsc-content-debt/)〉）。

## 常見問題（FAQ）

**Q1：title 和 H1 可以不一樣嗎？**
可以，Google 也確認兩者可分工——title 給搜尋結果頁（短、帶鉤子），H1 給進站讀者（可更完整）。但主題必須一致，差太多會提高被改寫機率。

**Q2：meta description 不影響排名，那可以不寫嗎？**
可以不寫，Google 會自動抓內文——但你就放棄了控制權。重要頁面（支柱頁、服務頁、高曝光文章）一定手寫；長尾農文可接受由「答案優先」第一段自動充當。

**Q3：關鍵字要在 title 出現幾次？**
一次就夠。出現兩次以上是堆砌訊號，反而觸發改寫。

**Q4：改了 title 多久生效？**
Google 重新爬取後即更新顯示，通常幾天到兩週。可用 GSC 網址檢查工具請求重新檢索加速。
